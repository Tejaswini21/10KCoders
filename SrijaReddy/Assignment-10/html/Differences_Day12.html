Differences Between ID , Class and Attribute Selectors

ID Selector Class Selector Attribute Selector
The id selector uses the id attribute of an HTML element to select a specific element The class selector selects HTML elements with a specific class attribute Attribute selectors are used to style HTML elements that have specific attributes or attribute values
syntax of id selector: id="value"; syntax of class selector: class="value"; syntax of attribute selector:[attr (~/^/$/*/|) = "value"];
ID selector starts with the "#" symbol followed by a unique name assigned to it. Class selector starts with "." followed by class name. Attribute selector starts with a specified element followed by [attribute]
Only one ID selector can be attached to an element. Multiple class selectors can be attached to an element. we have different types of attribute selectors for different puropose
ID is unique in a page and can only apply to at most one element The class can be applied to multiple elements so it could be multiple times on a single page we can apply different attribute selectors to an element
