Semantic Tags

In HTML semantics refers to the tags that provide meaning to an HTML page rather than just presentation.It makes HTML more comprehensible by better defining the different sections and layout of web pages.

  1. < p > : represents a paragraph in an HTML document.

  2. < h1 > to < h6 > :It defines headings for an HTML document from level 1 to level 6.

  3. < a > :It is termed as anchor tag and it creates a hyperlink or link.

  4. < ul > :It defines unordered list of items.

  5. < ol > :It defines an ordered list of items.

  6. < li > :It is used to represent items in list.

  7. < img > :It is used to insert an image within an HTML document.
