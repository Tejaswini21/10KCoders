Meta Tags

HTML < meta > tag is used to represent the metadata about the HTML document. It specifies page description, keywords, copyright, language, author of the documents, etc. The metadata does not display on the webpage, but it is used by search engines, browsers and other web services which scan the site or webpage to know about the webpage.

  1. < title >: it is used to provide title to our webpage

  2. < meta charser="UTF-8" > :UTF-8 is a character encoding capable of encoding all possible characters (called code points) in Unicode.

  3. < meat name="keywords" content="HTML, CSS, Javascript" >: it defines keywords for search engines

  4. < meta name="description" content="free tutorials for html and css" >:it is used to define description for your webpage

  5. < meta name="author" content="srija" >:it is used to define the author of the page

  6. < meta http-equiv="refresh" content="20" >: it is used to specify to refresh the document for every mentioned seconds

  7. < meta name="viewpoint" content=""width=device-width , initial-scale=1.0" >:it is used to set viepoint to make your website look good on all devices
